# LoginApp

## Overview
LoginApp is a simple Android application that demonstrates a user login flow. The application includes multiple screens, such as login, home, profile, and master user information screens.

## Setup
To get started with the LoginApp, follow the steps below:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Shafarulluqyan/LoginApp.git
   cd LoginApp
   ```

2. **Open the project in Android Studio:**
   - Start Android Studio.
   - Open the cloned project directory.

3. **Build the project:**
   - Click on `Build` in the menu bar.
   - Select `Make Project`.

4. **Run the application:**
   - Connect an Android device or start an emulator.
   - Click on `Run` in the menu bar.
   - Select `Run 'app'`.

## Project Structure
- **activities:** Contains the activity classes for different screens (LoginActivity, HomeActivity, ProfileActivity, MasterActivity).
- **models:** Contains the User model class.
- **utils:** Contains utility classes like BaseActivity.
- **res:** Contains resources such as layouts, drawable files, values, etc.

## Features
- **User Login:** Users can log in with their email and password.
- **Home Screen:** Displays a welcome message and buttons to navigate to the profile and master screens.
- **Profile Screen:** Displays the logged-in user's information.
- **Master Screen:** Displays a list of all users.
